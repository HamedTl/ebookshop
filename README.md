# 📚 EbookShop

A modern, responsive, and clean **online bookstore front-end template** built with **HTML**, **Tailwind CSS**, **Alpine.js**, and **Swiper.js**.

This project provides a professional UI for an online bookstore and can be integrated with any backend framework such as Laravel, Django, ASP.NET, Node.js, or others.

---

## ✨ Features

* 📱 Fully Responsive Design
* 🎨 Modern & Clean UI
* ⚡ Built with Tailwind CSS
* 🧩 Interactive Components using Alpine.js
* 🎠 Swiper.js Sliders
* 📚 Book Store Landing Page
* 🛍️ Product Listing Page
* 📰 Blog / Weblog Page
* 🇮🇷 RTL & Persian Layout
* 🔤 Persian Typography (YekanBakh Font)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/HamedTl/ebookshop.git
```

Go to the project directory:

```bash
cd ebookshop
```

Open the project using your preferred editor.

Since this is a static project, you can simply open `index.html` in your browser.

For a better development experience, use **VS Code Live Server** or any local web server.
